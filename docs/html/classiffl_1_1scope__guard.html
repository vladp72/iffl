Detailed Description

template<typename G>
class iffl::scope_guard< G >

template class that can be parametrised with a functor or a lambda that it will call in distructor.

Used to execute functor as we are leaving scope, unless it was explicitely disarmed. flat_forward_list uses this helper to deallocate memory on failures. scoped_guald inherits from the functor to minimize tyoe size when functor is emty type. For more information see Empty Base Class Optimization (EBCO).
